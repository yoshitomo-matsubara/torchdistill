torchdistill.losses.registry.register_low_level_loss(arg=None, **kwargs)[source]ÔÉÅ

Registers a low-level loss class or function to instantiate it.

Parameters:

arg (class or Callable or None) ‚Äì class or function to be registered as a low-level loss.

Returns:

registered low-level loss class or function to instantiate it.

Return type:

class or Callable

Note

The low-level loss will be registered as an option. You can choose the registered class/function by specifying the name of the class/function or key you used for the registration, in a training configuration used for torchdistill.core.distillation.DistillationBox or torchdistill.core.training.TrainingBox.

If you want to register the class/function with a key of your choice, add key to the decorator as below:

>>> from torch import nn
>>> from torchdistill.losses.registry import register_low_level_loss
>>>
>>> @register_low_level_loss(key='my_custom_low_level_loss')
>>> class CustomLowLevelLoss(nn.Module):
>>>     def __init__(self, **kwargs):
>>>         print('This is my custom low-level loss class')

In the example, CustomLowLevelLoss class is registered with a key ‚Äúmy_custom_low_level_loss‚Äù. When you configure torchdistill.core.distillation.DistillationBox or torchdistill.core.training.TrainingBox, you can choose the CustomLowLevelLoss class by ‚Äúmy_custom_low_level_loss‚Äù.

class torchdistill.losses.high_level.WeightedSumLoss(model_term=None, sub_terms=None, **kwargs)[source]ÔÉÅ

A weighted sum (linear combination) of mid-/low-level loss modules.

If model_term contains a numerical value with weight key, it will be a multiplier \(W_{model}\) for the sum of model-driven loss values \(\sum_{i} L_{model, i}\).

\[L_{total} = W_{model} \cdot (\sum_{i} L_{model, i}) + \sum_{k} W_{sub, k} \cdot L_{sub, k}\]
Parameters:

  • model_term (dict or None) ‚Äì model-driven loss module configurations.

  • sub_terms (dict or None) ‚Äì loss module configurations.

class torchdistill.losses.mid_level.KDLoss(student_module_path, student_module_io, teacher_module_path, teacher_module_io, temperature, alpha=None, beta=None, reduction='batchmean', **kwargs)[source]ÔÉÅ

A standard knowledge distillation (KD) loss module.

\[L_{KD} = \alpha \cdot L_{CE} + (1 - \alpha) \cdot \tau^2 \cdot L_{KL}\]

Geoffrey Hinton, Oriol Vinyals, Jeff Dean: ‚ÄúDistilling the Knowledge in a Neural Network‚Äù @ NIPS 2014 Deep Learning and Representation Learning Workshop (2014)

Parameters:

  • student_module_path (str) ‚Äì student model‚Äôs logit module path.

  • student_module_io (str) ‚Äì ‚Äòinput‚Äô or ‚Äòoutput‚Äô of the module in the student model.

  • teacher_module_path (str) ‚Äì teacher model‚Äôs logit module path.

  • teacher_module_io (str) ‚Äì ‚Äòinput‚Äô or ‚Äòoutput‚Äô of the module in the teacher model.

  • temperature (float) ‚Äì hyperparameter \(\tau\) to soften class-probability distributions.

  • alpha (float) ‚Äì balancing factor for \(L_{CE}\), cross-entropy.

  • beta (float or None) ‚Äì balancing factor (default: \(1 - \alpha\)) for \(L_{KL}\), KL divergence between class-probability distributions softened by \(\tau\).

  • reduction (str or None) ‚Äì reduction for KLDivLoss. If reduction = ‚Äòbatchmean‚Äô, CrossEntropyLoss‚Äôs reduction will be ‚Äòmean‚Äô.
